Footer Brand block

Footer Brand block example

The Footer Brand block is a mandatory core block that should be included in the footer section of all pages and views within your Carleton app. This block contains the Carleton logo, which acts as navigation to the university's homepage and is important for conveying trust and our brand. It also holds key legal links to our copyright, accessibility and privacy policies. Lastly, it contains the site's active RDS version to help identify if the code is current.
